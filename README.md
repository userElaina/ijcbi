# IJCBI

Because [**IJCAI**](https://www.ijcai.org/) was downgraded from a [CCF-A](https://ccf.atom.im/) conference to a CCF-B conference, it is jokingly referred to as **IJCBI**.

由于[中国计算机学会推荐国际学术会议和期刊目录](https://ccf.atom.im/)(2026)中 [**IJCAI**](https://www.ijcai.org/) 被降为 B 类会议, 故戏称 **IJCBI**.
